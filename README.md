## 👨‍💻 Installation 👨‍💻 
#### Step 1.
``` bash
git clone https://github.com/Didier-Lucu/Bamboo-Forest.git
```
#### Step 2.
``` bash
cd Bamboo-Forest
```
#### Step 3. 
``` bash
cd ucla-bamboo-forest
```
#### Step 4. 
``` bash
npm install
```
#### Step 5. 
``` bash
npm start
```
